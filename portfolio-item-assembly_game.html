Assembly game

front-end dev

📝 Description:

Assembly: Endgame is a minimalist word-guessing game where each wrong guess eliminates a programming language — until you're left with the last one standing: Assembly. Inspired by Hangman, this game challenges your logic, memory, and typing reflexes, wrapped in a playful coding-themed interface.

🛠️ GitHub repository

🎮 Game

🧩 Gameplay Features:

  • Guess a hidden word letter by letter
  • Each incorrect guess removes one programming language from the stack
  • You have 8 attempts to figure out the word before it's game over
  • Dynamic visual feedback and celebratory confetti on victory 🎉

🔧Tech Stack:

  • React with Hooks (useState)
  • Confetti animation using react-confetti
  • Dynamic class handling via clsx
  • Responsive design with semantic HTML and accessibility in mind
